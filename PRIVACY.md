# 데이터 처리 안내

SOOP AI 타임라인은 영상·오디오 원본 파일을 디스크에 저장하지 않습니다. 공개 다시보기는 오디오 전용 스트림을 메모리에서 처리하고, 라이브는 저화질 스트림에서 오디오만 해독합니다.

음성 인식은 사용자의 PC에서 faster-whisper로 수행됩니다. 다음 정보는 타임라인 주제 판정을 위해 사용자가 설정한 Google Gemini API로 전송됩니다.

- 타임스탬프가 포함된 음성 인식 자막
- 영상 제목과 스트리머 표시 이름
- 사용자가 등록한 스트리머 단어 사전
- 이전 구간의 타임라인 후보

Gemini API 키는 운영체제의 자격 증명 저장소에 보관합니다. 프로그램 데이터 폴더에는 다음 항목이 저장될 수 있습니다.

- Whisper 자막과 중간 체크포인트
- Gemini 구간별 중간 결과
- 검수 중인 타임라인과 버전 기록
- 스트리머 목록, 설정, 최근 오류 로그

설정에서 전체 자막 캐시와 중간 결과를 삭제할 수 있고, 각 영상 편집 화면에서도 해당 영상의 캐시만 삭제할 수 있습니다. 스트리머를 삭제하면 연결된 VOD 기록과 로컬 캐시도 함께 삭제됩니다.

이 프로그램은 사용자가 처리 권한을 보유했거나 해당 이용이 허용된 영상에 사용해야 합니다. 공개 열람 가능 여부만으로 자동 분석·외부 AI 전송·댓글 게시 권한이 부여되는 것은 아닙니다. SOOP 약관, 스트리머의 의사, 저작권 및 개인정보 관련 의무는 사용자가 별도로 확인해야 합니다.
